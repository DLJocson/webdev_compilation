Introduction to HTML

HTML, or HyperText Markup Language, is the standard language used to create and design web pages. It provides the structure and content of a webpage, allowing developers to organize text, images, links, and other media in a way that browsers can interpret and display.

HTML uses a system of tags to define elements on a webpage. These tags are enclosed in angle brackets (< >) and typically come in pairs: an opening tag (e.g., <div>) and a closing tag (e.g., </div>). The content between these tags is what gets displayed on the webpage.

  • Hypertext: Refers to the way in which Web pages are linked together.
  • Markup Language: Means you simply "mark-up" a text document with tags that tell a Web browser how to structure and display it.
In HTML, adding extra lines of code does not affect the output of the webpage. This is because HTML ignores whitespace, including spaces, tabs, and newlines. You need to use specific tags to create line breaks or paragraphs.

⚙️ How HTML Works (The Workflow)

  1. Writing the Code: Either through a text editor or an Integrated Development Environment (IDE).
  2. Saving the File: With a .html extension.
  3. Opening in a Browser: The browser reads the HTML code and renders the webpage accordingly.

Basic Elements of an HTML

Every HTML document has a basic structure that includes the following elements:

  • <!DOCTYPE html>: Declares the document type and version of HTML being used.
  • <html>: The root element that wraps all content on the page.
  • <head>: Contains meta-information about the document, such as the title and links to stylesheets.
  • <body>: Contains the content that is displayed on the webpage.

These elements all contain a start tag and an end tag. With elements being able to nest within each other.

Anatomy of an HTML Code 

<!DOCTYPE html>
<html>
    <head> --> Start tag for the head section
        <title>My First Webpage</title>
    </head>  --> End tag for the head section
    <body>
        <h1>Hello, World!</h1>
        <p>This is my first webpage.</p>
    </body>
</html>

🏗️ Core HTML Structures

1. Headings, Paragraphs, & Breaks

  • Headings (<h1> to <h6>): Defines the document outline. <h1> is the most important, and make sure to use them in a hierarchical order.
  • Paragraphs (<p>): Starts on a new line; browsers automatically add whitespace before and after.
  • Line Break (<br>): An empty tag used to force a new line without starting a new paragraph.
  • Horizontal Rule (<hr>): Defines a thematic break (usually displayed as a line) to separate content.

2. Preformatted Text (<pre>)

The <pre> tag is used to display preformatted text. It preserves both spaces and line breaks, making it ideal for displaying code or any text where formatting is important.


<pre>
My Bonnie lies over the ocean.
    My Bonnie lies over the sea.
</pre>

3. HTML Links

The anchor tag (<a>) creates hyperlinks, which are the foundation of the web.

  • Internal: <a href="#section-id"> (Jumps to a section on the same page)
  • External: <a href="https://example.com"> (Goes to another website)
  • Email: <a href="mailto:user@email.com"> (Opens email client)
  • Phone: <a href="tel:+639171234567"> (Dials number on mobile)
  • Download: <a href="file.pdf" download> (Downloads instead of navigating)

Embedding Multimedia

HTML allows you to embed various types of media, such as images, audio, and video, using specific tags.

1. Images (<img>)

The <img> tag is used to embed images. It is an empty tag, meaning it does not have a closing tag. The most important attributes for the <img> tag are:

  • src: Specifies the path to the image file.
  • alt: Provides alternative text for the image if it cannot be displayed.
  • width & height: Define the dimensions of the image.

2. Audio (<audio>)

The <audio> tag is used to embed audio files. It can contain one or more <source> elements to specify different audio formats for compatibility across browsers. The controls attribute adds playback controls to the audio player.

3. Video (<video>)

The <video> tag is used to embed video files. Similar to the <audio> tag, it can contain multiple <source> elements for different video formats. The controls attribute adds playback controls to the video player.

Sample Code for Embedding Multimedia 

<!-- Image Example -->
<figure>
    <img src="image.jpg" alt="Description of image" width="500" height="600" 
    title="Sample Image" loading="lazy">
    <figcaption>Figure 1: Example of an embedded image in HTML.</figcaption>
</figure>


<!-- Audio Example -->
<audio controls preload="metadata">
    <source src="audio.mp3" type="audio/mpeg">
    <source src="audio.ogg" type="audio/ogg">
    Your browser does not support the audio element.
</audio>


<!-- Video Example -->
<video controls width="400" poster="thumbnail.jpg" preload="metadata">
    <source src="video.mp4" type="video/mp4">
    <source src="video.webm" type="video/webm">

    <!-- Caption Track -->
    <track src="subtitles.vtt" kind="subtitles" srclang="en" label="English">

    Your browser does not support the video tag.
</video>

💡 Quick Facts

  • HTML was created by Sir Tim Berners-Lee in late 1991 but wasn't officially published until 1993.
  • HTML5 is the latest major version, adding semantic elements and native multimedia tags.
  • Tags are generally not case-sensitive, but using lowercase is the industry standard.
